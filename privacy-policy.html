Política de privacidade

Importante!

O Robby ou qualquer membro da equipe JAMAIS irá te pedir qualquer dado pessoal, como Idade, telefone, dados do cartão de crédito, sua senha do Discord, etc. Caso receba, nunca informe nenhum dado pessoal, pois certamente, é um SCAM, reporte o usuário para o Discord e para nossa equipe!

Antes de utilizar o Robby, não se esqueça de ler as diretrizes e os Termos de Uso (TOS) do Discord

Quais informações o Robby coleta?

  • Comandos
    Quando um usuário utiliza um comando, o Robby guarda e registra por um curto período a mensagem e o ID de quem executou um certo comando, Registramos isso para caso um usuário esteja utilizando os mesmos com más intenções ou caso o usuário tente ficar 'spammando' comandos com a intenção de prejudicar e/ou derrubar o Robby.
  • Informações fornecidas pela API do Discord
    O Robby coleta e utiliza no Discord dados que são fornecidos pela própria API do Discord, dados como IDs de usuários e de guilds.
  • Suas informações dentro do Robby
    O Robby também registra informações que são criadas pelo próprio usuário e informações dele dentro do bot, como os comandos de economia, por exemplo, o daily, e o atm, no qual ele registra informações como a sua quantidade de robby coins, sua posição no rank global e local, e dados do `/perfil` como por exemplo, o seu sobremim.

Todos os dados que são coletados citados acima, são necessários para um bom funcionamento do Robby. Todas elas estão de acordo com a política dos desenvolvedores do Discord
